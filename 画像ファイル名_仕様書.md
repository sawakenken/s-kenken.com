# s-kenken.com 画像ファイル名 仕様書

`images/` の中に下記の名前で画像を入れてください（半角英数字・小文字、日本語/スペースNG）。
写真が無い枠は自動でグラデ表示になるので、用意できたものから順にアップでOK。
推奨：長辺2000px / JPEG / 各1MB以下。
※ クライアントロゴだけは別仕様（背景透過PNG）。詳細は「7. クライアントロゴ」を参照。

```
images/
├─ hero/
├─ gallery/<カテゴリ>/
├─ works/
├─ clients/
└─ about/
```

## 1. ヒーロー（トップ最上部）

※ `-pc` はデスクトップ表示用。未アップなら縦写真にフォールバック。
